## ⚙️ Setup Instructions (Do This Once)

### 1. Install Python
- Download from https://python.org (version 3.9 or higher)
- During install, tick ✅ "Add Python to PATH"

### 2. Install MySQL
- Download MySQL Community Server: https://dev.mysql.com/downloads/installer/
- Remember your root password — you'll need it

### 3. Create a Project Folder
```
mkdir debt_project
cd debt_project
```

### 4. Install Required Python Libraries
```bash
pip install -r requirements.txt
```

### 5. Get the Dataset
- Go to: https://www.kaggle.com/datasets (search "International Debt Statistics World Bank")
- Download the CSV file
- Rename it to `international_debt.csv`
- Place it inside the `data/` folder

## 🚀 How to Run — Step by Step

Run each script IN ORDER:

### Step 1: Clean the Data
```bash
python step1_clean_data.py
```
- Input:  `data/international_debt.csv`
- Output: `data/cleaned_debt.csv`

### Step 2: Exploratory Data Analysis
```bash
python step2_eda.py
```
- Input:  `data/cleaned_debt.csv`
- Output: 6 PNG chart files in `charts/` folder

### Step 3: Set Up MySQL Database
Open `step3_database.py` and change line:
```python
"password": "your_password"   ← change to YOUR MySQL password
```
Then run:
```bash
python step3_database.py
```
- Creates database `debt_db` with 3 tables
- Inserts all cleaned data into MySQL

### Step 4: Run SQL Queries
**Option A — MySQL Workbench (recommended for learning):**
1. Open MySQL Workbench
2. File → Open SQL Script → select `step4_queries.sql`
3. Run each query by clicking inside it and pressing Ctrl+Enter

**Option B — From Python:**
```bash
python step4_run_queries.py
```
- Output: `outputs/query_results.txt`

### Step 5: Launch Dashboard
```bash
streamlit run step5_dashboard.py
```
- Opens browser at http://localhost:8501
- Use the sidebar to filter countries and indicators
- All charts update automatically

## 🗄️ Database Schema

```
┌──────────────────┐       ┌──────────────────────┐       ┌─────────────────────┐
│   countries      │       │     debt_data         │       │    indicators       │
│──────────────────│       │──────────────────────│       │─────────────────────│
│ country_id  (PK) │◄──────│ country_id    (FK)   │──────►│ indicator_id   (PK) │
│ country_name     │       │ indicator_id  (FK)   │       │ indicator_name      │
│ country_code     │       │ debt                 │       │ indicator_code      │
└──────────────────┘       └──────────────────────┘       └─────────────────────┘
```

**Why 3 tables?**
- Instead of storing "China" 400 times, we store ID=1 once
- Saves space, prevents typos, faster queries
- This is called "database normalization"

## 🔑 Key Concepts for Beginners

| Concept | What It Means |
|---------|--------------|
| DataFrame | A table in Python (like Excel) |
| NaN / null | A missing/empty value |
| GROUP BY | Group rows and calculate totals per group |
| JOIN | Combine two tables using a shared ID column |
| Primary Key | Unique ID for every row in a table |
| Foreign Key | A column linking to another table's PK |
| Window Function | Calculations across related rows (RANK, SUM OVER) |

## ❓ Common Errors & Fixes

| Error | Fix |
|-------|-----|
| `ModuleNotFoundError: pandas` | Run `pip install pandas` |
| `Access denied for user 'root'` | Wrong MySQL password in DB_CONFIG |
| `No such file: cleaned_debt.csv` | Run step1 first |
| `streamlit: command not found` | Run `pip install streamlit` |
| `FileNotFoundError: international_debt.csv` | Put your CSV in the data/ folder |
